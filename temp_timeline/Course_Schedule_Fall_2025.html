Intro to Software Engineering Course Schedule (Fall 2025)

Week Dates Topics Lectures/Details In-class Work Homework/Assignments Reading (Before Class) Project Milestones
1 Aug 18–24 Introduction to Software Engineering Course overview, importance, historical context, SE vs. programming. Note: Classes start Aug 20. Chapter 1: Introduction (Historical Context, Topics of Study)
2 Aug 25–31 SDLC and Agile Scrum SDLC overview (Waterfall, Agile, XP), Agile principles, Scrum intro. Simulate a Scrum planning session. Read Chapter 2; compare Waterfall vs. Agile. Chapter 2: Processes (Agile Manifesto, XP, Scrum)
3 Sep 1–7 Version Control with Git and GitHub Git basics, GitHub essentials. Note: Labor Day (Sep 1). Hands-on Git exercises: commit, branch. Create a GitHub repo and submit a simple project. Appendix: Git
4 Sep 8–14 Project Management Board, Scrum (Advanced) Scrum roles, ceremonies, artifacts, Zenhub; advanced Git. Zenhub demo and setup for teams; merge exercises. Set up Zenhub for a mock project. Chapter 2: Processes (Scrum, Kanban) Milestone 1: Team formation, product backlog, first draft planning, Git/Zenhub setup.
5 Sep 15–21 Requirements Engineering I High-level analysis, low-level analysis (stories, use cases, use case diagram, functional and non-functional requirements, SRS). Write user stories for a sample app. Draft SRS for a simple system. Chapter 3: Requirements (User Stories, Use Cases) Progress check: Update Zenhub with stories.
6 Sep 22–28 Requirements Engineering II MVP, A/B testing, planning poker, UML intro. Create class and use case diagrams. Create class and use case diagrams. Practice planning poker on sample stories. Chapter 3: Requirements (MVP, A/B Testing) Milestone 2: SRS, MVP with console, user stories, use case diagram, basic test cases.
7 Sep 29–Oct 5 UI Design and Architecture GUI concepts, wireframes, mockup, prototype, architectures (Layered, MVC). Note: Asynchronous Midterm Exam this week. Design wireframes for a UI. Read Chapter 7; diagram an MVC app. Chapter 7: Architecture (MVC, Microservices) Progress check: Propose wireframes.
8 Oct 6–12 Design Principles Conceptual integrity, hiding, cohesion, coupling, SOLID. Apply SOLID to code examples. Refactor code to follow SOLID. Chapter 5: Design Principles (Conceptual Integrity, Information Hiding, Cohesion, Coupling, SOLID) Milestone 3: UML diagrams (class, activity, sequence), MVC code setup, wireframes, basic tests.
9 Oct 13–19 Design Patterns I Factory, Singleton, Observer, Strategy. Note: Fall Break (Oct 16–18). Implement a pattern in Python. Read Chapter 6; code examples. Chapter 6: Design Patterns (Factory, Singleton, Observer, Strategy) Progress check: Identify patterns for features.
10 Oct 20–26 Design Patterns II Adapter, pattern review and application. Group pattern application to projects. Extend homework with more patterns. Chapter 6: Design Patterns (Adapter, etc.) Milestone 4: Adopt 3–7 features, design principles/patterns, test cases.
11 Oct 27–Nov 2 Testing Unit, integration testing, TDD, tools, test coverage. Write unit tests for sample code. Read Chapter 8; test a small app. Chapter 8: Testing (Unit Tests, Coverage, TDD) Progress check: Add tests to code.
12 Nov 3–9 Refactoring and DevOps Refactoring, code smells, DevOps, CI/CD. Refactor code snippets. Set up basic CI pipeline. Chapter 9: Refactoring; Chapter 10: DevOps (CI/CD) Milestone 5.1: Test cases + coverage.
13 Nov 10–16 Project Presentation and Kanban Project presentations, intro to Kanban. Presentations and feedback. Reflection on Kanban vs. Scrum. Chapter 2: Processes (Kanban) Milestone 5.2: Presentation, retrospective.
14 Nov 17–23 Critical Revision and Agile in Practice Review UML, Agile in practice. Group review of key concepts. Chapter 4: Models (UML Diagrams) Final submission: Address issues.
15 Nov 24–30 Course Review Review key concepts, prepare for final exam. Note: Thanksgiving Break (Nov 24–29). Asynchronous review quiz. Complete review quiz. Review Chapters 1–10
16 Dec 1–7 (Finals: Dec 8–12) Final Exam (Scheduled during finals week, Dec 8–12)
